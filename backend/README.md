# AksesKota Backend

Folder ini disiapkan untuk REST API berbasis Express.

Sebelum implementasi, baca kontrak dan kebutuhan sistem di `../docs/BACKEND_HANDOFF.md`.

Rencana struktur awal:

```text
backend/
├── src/
│   ├── controllers/
│   ├── middleware/
│   ├── routes/
│   ├── services/
│   └── app.js
├── tests/
├── .env.example
└── package.json
```
